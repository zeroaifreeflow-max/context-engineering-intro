# วิธีใช้งาน — CE Starter Kit (โปรเจคใหม่)

## ขั้นตอน

1. **Copy** โฟลเดอร์นี้ทั้งหมดไปวางที่ root ของโปรเจคใหม่
2. **เปิดโปรเจค**ใน AI Editor (Claude Code, Cursor, Windsurf) แล้วสั่ง:
   ```
   อ่าน KICKSTART.md แล้วทำตาม
   ```
3. **ตอบคำถาม AI** เรื่อง tech stack, architecture, coding style ของคุณ
4. **ตรวจสอบไฟล์**ที่ AI สร้างให้ แก้ไขถ้าจำเป็น

## ไฟล์ที่ได้

- `CLAUDE.md` = กฎที่ AI เชื่อฟังทุกบทสนทนา (style, testing, conventions)
- `PLANNING.md` = เอกสาร architecture, tech stack, design decisions
- `TASK.md` = ระบบ track งานและ TODOs
- `INITIAL.md` = Template สำหรับขอฟีเจอร์ใหม่ผ่าน PRP workflow

AI จะลบ `KICKSTART.md` และ `templates/` ออกให้เองหลังเสร็จ — พร้อมใช้งานทันที

> ยิ่งตอบคำถาม AI ละเอียดเท่าไหร่ ไฟล์ context ยิ่งแม่นยำ → โค้ดที่ได้ยิ่งตรงตามมาตรฐาน
