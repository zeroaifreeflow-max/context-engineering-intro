# วิธีใช้งาน — CE Starter Kit (Optimize โปรเจคเก่า)

## ขั้นตอน

1. **Copy** โฟลเดอร์นี้ทั้งหมดไปวางที่ root ของโปรเจคเก่า
2. **เปิดโปรเจค**ใน AI Editor (Claude Code, Cursor, Windsurf) แล้วสั่ง:
   ```
   อ่าน KICKSTART.md แล้วทำตาม
   ```
3. **รอ AI สแกน codebase** — AI วิเคราะห์โค้ด, โครงสร้าง, conventions ให้เองอัตโนมัติ
4. **ยืนยันผลสแกน** ที่ AI สรุปมา แก้ไขถ้ามีอะไรผิดหรือขาดหาย

## ไฟล์ที่ได้

- `CLAUDE.md` = กฎสำหรับ AI อิงจาก patterns จริงในโค้ดของคุณ
- `PLANNING.md` = เอกสาร architecture อิงจากโครงสร้างจริงของโปรเจค
- `TASK.md` = ระบบ track งาน + Improvement Backlog ที่ AI ตรวจพบ
- `INITIAL.md` = Template สำหรับขอฟีเจอร์ใหม่ผ่าน PRP workflow

AI จะลบ `KICKSTART.md` และ `templates/` ออกให้เองหลังเสร็จ — พร้อมใช้งานทันที

> Kit นี้ไม่แตะโค้ดเดิม — แค่สร้างไฟล์ context เพื่อให้ AI เข้าใจโปรเจคของคุณ
